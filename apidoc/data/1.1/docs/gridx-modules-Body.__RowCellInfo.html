gridx/modules/Body.__RowCellInfo (version 1.1)

This structure includes all possible information that can be used to identify a row or a cell, it is used to retrieve a row or a cell in grid body. Usually user only need to provide some of them that is sufficient to uniquely identify a row or a cell, e.g. rowId, or rowIndex and parentId, or visualIndex.

Note:This is not a real constructor, but just a description of the type of object that should be passed as a parameter to some method(s), and/or the return value from some method(s). In other words, the type exists only for documentation purposes, and you cannot call new Body.__RowCellInfo().

Property Summary

  • colId The ID of a column (should not be false values)
  • colIndex The index of a column.
  • rowId The ID of a row.
  • rowIndex The index of a row.
  • visualIndex The visual index of a row.

The index of a row. It is the index below the parent of this row. The parent of root rows is an imaginary row with id "" (empty string).

visualIndex
Defined by gridx/modules/Body

The visual index of a row. It represents the visual position of this row in the current body view. If there are no pagination, no filtering, no tree structure data, this value is equal to the row index.
