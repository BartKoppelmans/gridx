gridx/support/LinkSizer (version 1.1)

gridx/support/_LinkPageBase

This is a grid bar plugin that can be used to switch page sizes using link buttons.

Property Summary

  • grid The grid widget this plugin works for.
  • sizes An array of available page sizes.
  • sizeSeparator

Method Summary

  • refresh() Refresh the UI using current arguments.
