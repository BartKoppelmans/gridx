gridx/mobile/Sort (version 1.1)

Provide sort api for the grid and provide a single UI in the header like traditional grid sorting.

Method Summary

  • _buildHeader() Add click to sort capability for header, only for single sort.
  • sort(args) Sort the grid by args which in store sorting format.
  • updateSortIndicators() Update the sorting indicators.
