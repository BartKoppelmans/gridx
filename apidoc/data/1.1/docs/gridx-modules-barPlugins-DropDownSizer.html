gridx/modules/barPlugins/DropDownSizer (version 1.1)

This grid bar plugin is to switch page sizes using select widget.

Property Summary

  • grid The grid widget this plugin works for.
  • sizerClass The constructor of the select widget
  • sizerProps The properties passed to select widget when creating it.
  • sizes An array of available page sizes.
