gridx/support/exporter/exporter (version 1.1)

Go through the grid using the given args and writer implementation. Return a dojo.Deferred object. Users can cancel and see progress of the exporting process. Pass the exported result to the callback function of the Deferred object.

Usage

exporter(grid, writer, args);
ParameterTypeDescription
gridgridx/Grid

The grid instance to export

writer__Writer

The writer implementation.

args__ExportArgs

Export arguments.

Returns: any

A deferred object.
