gridx/modules/Menu.__MenuArgs (version 1.2)

Note:This is not a real constructor, but just a description of the type of object that should be passed as a parameter to some method(s), and/or the return value from some method(s). In other words, the type exists only for documentation purposes, and you cannot call new Menu.__MenuArgs().

Property Summary

  • hookPoint Indicates from where the menu should occur.
  • selected Indicates whether to bind this menu only to the selected items.

Properties

hookPoint
Defined by gridx/modules/Menu

Indicates from where the menu should occur. One of "cell", "headercell", "row", "header", "body", "grid". If invalid, default to "grid".

selected
Defined by gridx/modules/Menu

Indicates whether to bind this menu only to the selected items. Default is false.
