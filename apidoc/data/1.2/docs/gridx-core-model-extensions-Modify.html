gridx/core/model/extensions/Modify (version 1.2)

  • Summary Enable model to change data without affecting the store.

Properties

Summary
Defined by gridx/core/model/extensions/Modify

Enable model to change data without affecting the store. All the changes will be saved in the modify extension. The byId and byIndex function will be wrapped in this extension.

Methods

clearLazyDat()
Defined by gridx/core/model/extensions/Modify

Undo all. Clear undo list. The initial name of this function is 'clear'. When use grid.model.clear(), this function won't be run because there is a function named 'clear'in ClientFilter. So rename this function to clearLazyData which is more in detail about what this function really do.

getChanged()
Defined by gridx/core/model/extensions/Modify

Get all the changed rows Ids.

Returns: any | Array

An array of changed row IDs.

isChanged()
Defined by gridx/core/model/extensions/Modify

Check whether a field is changed for the given row.

Returns: any | boolean

True if it does get changed.

redo()
Defined by gridx/core/model/extensions/Modify

redo next edit change.

Returns: any | boolean

True if successful, false if nothing to redo.

save()
Defined by gridx/core/model/extensions/Modify

write to store. Clear undo list.

Returns: any

A Deferred object indicating all the store save operation has finished.

set()
Defined by gridx/core/model/extensions/Modify

Set some fields in a row. Can set multiple fields altogether. This is one single operation.

undo()
Defined by gridx/core/model/extensions/Modify

Undo last edit change.

Returns: any | boolean

True if successful, false if nothing to undo.
