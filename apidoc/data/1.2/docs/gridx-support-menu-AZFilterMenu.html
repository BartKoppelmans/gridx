gridx/support/menu/AZFilterMenu (version 1.2)

gridx/support/menu/_FilterMenuBase

This is a sample implementation for the alphabetic filter, default to ['A-F', 'G-L', 'M-R', 'S-Z'].

Property Summary

  • alphabeticRanges The ranges of alphabetic filter.
  • colId The column id the menu attached to.
  • grid The grid widget this plugin works for.
  • leftClickToOpen Whether left or right mouse click to open the menu.

Method Summary

  • bindGrid(grid, col) When the menu is created, bind the grid to the menu.
