gridx/core/Cell (version 1.2)

Represents a cell of a grid

An instance of this class represents a grid cell. This class should not be directly instantiated by users. It should be returned by grid APIs.

Property Summary

  • column
  • grid Reference to the grid
  • model Reference to this grid model
  • row Reference to the row of this cell

Method Summary

  • applyEdit() Apply change to store for this cell
  • beginEdit() Begin editing mode on this cell
  • cancelEdit() Cancel editing mode on this cell.
  • contentNode() Get the dom node in this cell that actually contains data.
  • data() Get the grid data of this cell.
  • deselect() Deselect this cell.
  • down() Get the below cell.
  • editor() Get the editor in this cell if it is in editing mode.
  • isEditing() Check whether this cell is in editing mode.
  • isSelected() Whether this cell is selected.
  • next(crossRow) Get the right (left if RTL) cell.
  • node() Get the dom node of this cell.
  • prev(crossRow) Get the left (right if RTL) cell.
  • rawData() Get the store data of this cell.
  • select() Select this cell.
  • setRawData(rawData) Set new raw data to this cell.
  • up() Get the above cell.
  • widget() Get the cell widget in this cell.

Defined by gridx/modules/Traverse

Get the right (left if RTL) cell. If crossRow is true, and the current cell is the last cell of a row, then return the first cell of the below row.

crossRowBoolean
