gridx/core/Row (version 1.2)

Represents a row of a grid

An instance of this class represents a grid row. This class should not be directly instantiated by users. It should be returned by grid APIs.

Property Summary

  • grid Reference to the grid
  • id The ID of this row
  • model Reference to this grid model

Method Summary

  • cell(column, isId) Get a cell object in this row
  • cells(start, count) Get cells in this row.
  • data() Get the grid data in this row.
  • deselect() Deselect this row.
  • down() Get the below row.
  • firstCell() Get the first cell of this row.
  • index() Get the index of this row
  • isSelectable() Check whether this row is selectable.
  • isSelected() Whether this row is selected.
  • item() Get the store item of this row
  • lastCell() Get the last cell of this row.
  • moveTo(target, skipUpdateBody)
  • nextSibling() Get the next sibling row under the same parent.
  • node() Get the dom node of this row.
  • parent() Get the parent row of this row.
  • prevSibling() Get the previous sibling row under the same parent.
  • rawData() Get the store data in this row.
  • root() Get the root ancestor row.
  • select() Select this row.
  • setRawData(rawData) Set new raw data of this row into the store
  • up() Get the above row.
  • visualIndex() Get the visual index (the position in current grid body) of this row.

cell(column, isId)
Defined by gridx/core/Row

Get a cell object in this row

ParameterTypeDescription
columngridx.core.Column | Integer | String

Column index or column ID or a column object

isIdBoolean
Optional.

If the column parameter is a numeric ID, set this to true

Returns: any

If the params are valid return the cell object, else return null.

cells(start, count)
Defined by gridx/core/Row

Get cells in this row.

ParameterTypeDescription
startInteger
Optional.

The column index of the first cell in the returned array. If omitted, defaults to 0, so row.cells() gets all the cells.

countInteger
Optional.

The number of cells to return. If omitted, all the cells starting from column 'start' will be returned.

Returns: any

An array of cells in this row

data()
Defined by gridx/core/Row

Get the grid data in this row.

Grid data means the result of the formatter functions (if exist). It can be different from store data (a.k.a. raw data).

Returns: any

An associative array using column IDs as keys and grid data as values
