gridx/support/QuickFilter (version 1.2)

Quick filter box.

Property Summary

  • autoApply If true, the filter will be applied to grid during typing in the filter box.
  • delay The time (in ms) delay before applying the filter after each key stroke in the filter box.
  • grid The grid widget this plugin works for.

Properties

autoApply
Defined by gridx/support/QuickFilter
