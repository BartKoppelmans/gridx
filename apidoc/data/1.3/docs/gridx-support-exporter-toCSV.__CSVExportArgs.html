gridx/support/exporter/toCSV.__CSVExportArgs (version 1.3)

Note:This is not a real constructor, but just a description of the type of object that should be passed as a parameter to some method(s), and/or the return value from some method(s). In other words, the type exists only for documentation purposes, and you cannot call new toCSV.__CSVExportArgs().

  • filter A predicate function (returns Boolean) to judge whether to export a row.
  • formatter
  • newLine The new line string used in CSV.

Properties

columns
Defined by gridx/support/exporter/exporter

An array of column ID. Indicates which columns to be exported. If invalid or empty array, export all grid columns.

count
Defined by gridx/support/exporter/exporter

Indicates the count of rows export. If invalid, export all rows up to the end of grid.

filter
Defined by gridx/support/exporter/exporter

A predicate function (returns Boolean) to judge whether to export a row. This constraint is applied upon the result rows of the selectedOnly parameter, if provided.

formatter
Defined by gridx/support/exporter/exporter
newLine
Defined by gridx/support/exporter/toCSV

The new line string used in CSV. Deault to '\r\n';

omitHeader
Defined by gridx/support/exporter/exporter

Indicates whether to export grid header. Default to false.

progressStep
Defined by gridx/support/exporter/exporter

Number of rows in each progress. Default to 0 (invalid means only one progress). After each progress, the deferred.progress() is called, so the exporting process can be viewed by the user.

selectedOnly
Defined by gridx/support/exporter/exporter

Whether only export selected rows. This constraint is applied upon the rows specified by start and count parameters. This paramter and the filter parameter can be used togetther. Default to false.

separator
Defined by gridx/support/exporter/toCSV
seperator
Defined by gridx/support/exporter/toCSV

The seperator string used in CSV. Default to comma ','.

start
Defined by gridx/support/exporter/exporter

Indicates from which row to start exporting. If invalid, default to 0.

useStoreData
Defined by gridx/support/exporter/exporter

Indicates whether to export grid data (formatted data) or store data. Default to false.
