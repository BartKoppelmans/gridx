gridx/modules/Edit.__ColumnDefinition (version 1.3)

Column definition parameters defined by Edit.

Note:This is not a real constructor, but just a description of the type of object that should be passed as a parameter to some method(s), and/or the return value from some method(s). In other words, the type exists only for documentation purposes, and you cannot call new Edit.__ColumnDefinition().

Property Summary

  • alwaysEditing If true then the cells in this column will always be in editing mode.
  • editable If true then the cells in this column will be editable.
  • editor Set the dijit/widget to be used when a cell is in editing mode.
  • editorArgs

Method Summary

  • canEdit(cell) Decide whether a cell is editable.
  • customApplyEdit(cell, value) If editing a cell is not as simple as setting a value to a store field, custom logic can be put here.
  • getEditorConnects(editor, cell)
  • initializeEditor(editor, cell) Do special initialization for the current cell.
  • uninitializeEditor(editor, cell) Called every time a cell widget is reused to a cell.

Event Summary

  • onEditorCreated(editor, column) Fired when an editor is created.

Properties

alwaysEditing
Defined by gridx/modules/Edit

If true then the cells in this column will always be in editing mode. Default is false.

editable
Defined by gridx/modules/Edit

If true then the cells in this column will be editable. Default is false.

editor
Defined by gridx/modules/Edit

Set the dijit/widget to be used when a cell is in editing mode. The default dijit is dijit.form.TextBox. This attribute can either be the declared class name of a dijit or the class construct function of a dijit (the one that is used behide "new" keyword).

editorArgs
Defined by gridx/modules/Edit

Methods

canEdit(cell)
Defined by gridx/modules/Edit

Decide whether a cell is editable. This makes it possible to config some cells to be uneditable in an edtibale column.

ParameterTypeDescription
cellgridx.core.Cell

The cell object

Returns: any

True if this cell should be editable, false if not.

customApplyEdit(cell, value)
Defined by gridx/modules/Edit

If editing a cell is not as simple as setting a value to a store field, custom logic can be put here. For example, setting multiple fields of store for a formatted cell. Can return a Deferred object if the work can not be done synchronously.
