gridx/modules/Bar.__BarItem (version 1.3)

Configurations for a in the grid bar. This configuration object will also be passed to the constructor of the plugin, if "pluginClass" is used, so the plugin-related parameters can also be declared here.

Note:This is not a real constructor, but just a description of the type of object that should be passed as a parameter to some method(s), and/or the return value from some method(s). In other words, the type exists only for documentation purposes, and you cannot call new Bar.__BarItem().

plugin
Defined by gridx/modules/Bar

A plugin instance. If this parameter exists, the "pluginClass" and "content" parameter will be ignored. Note that if declared in this way, the same instance can not be used in other bar positions. If there's no other configurations, and this plugin has "domNode" attribute, then this instance can be used directly in place of the whole __BarItem configuration object.

pluginClass
Defined by gridx/modules/Bar

The class name (MID) or contructor of the plugin. The plugin instance will be created automatically. If there's no other configurations, this class name/constructor can be used directly in place of the whole __BarItem configuration object. If this parameter exists, the "content" parameter will be ignored.
