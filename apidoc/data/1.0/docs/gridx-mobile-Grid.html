gridx/mobile/Grid (version 1.0)

A mobile grid that has fixed header, footer and a scrollable body.

Property Summary

  • autoHeight If true, it's must be a children of dojox.mobile.View and it occupies the rest height of the screen.
  • columns Column definition to show the grid from store
  • hScroller Whether to show the horizontal scroller
  • showHeader Whether to show the grid header
  • vScroller Whether to show the virtical scroller

Method Summary

  • _buildBody(items) Build the grid body
  • _buildHeader() Build the grid header when showHeader is true.
  • _createRow(item) Create a grid row by object store item.
  • _getCellContent(col, item) Get a cell content by the column definition. * Currently only support string content, will add support for widget in future.
  • buildGrid() Build the whole grid
  • buildRendering() Build the grid dom structure.
  • refresh() Fetches the data and generates the list items.
  • resize() Calculate the height of grid body according to the autoHeight property.
  • setColumns(columns) Set columns to show for the grid.
  • startup()

Event Summary

  • onComplete(items) An handler that is called after the fetch completes.
  • onDelete(item, removedFrom) Deletes an existing item.
  • onError(errorData) An error handler.
  • onUpdate(item, insertedInto) Adds a new item or updates an existing item.
