GridX API

Welcome to the GridX API documentation. You are browsing version 1.0 of GridX. Use the selector to the left to change versions.

To find the object you're interested in, use the tree to the left...or use the quick links below. To print the documentation, simply navigate to the object of your choice and hit Print.

Main Classes of GridX

Grid
Pure GridX with only core modules.

Modules of GridX

VirtualVScroller
Provide row lazy-rendering during vertical scrolling. Suitable for large data set.
FilterBar
A filter bar on top of grid, can show filter dialog when clicked on it.
CellWidget
Show widgets in grid cell. Reuse these widgets across rows to improve efficiency.
extendedSelect/Row
Select row using mouse or keyboard. Support swipe-selection, CTRL-selection, and SHIFT-selection.
IndirectSelect
Provide a check-box/radio-button column for row selection.
dnd/Row
Drag-and-Drop rows within grid, across grid or to/from other widgets.
Tree
Tree grid.
ColumnLock
Lockup leading columns.
PaginationBar
Paging Grid. An alternative to the virtual scroller.
