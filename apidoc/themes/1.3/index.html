GridX API

Welcome to the GridX API documentation. You are browsing version 1.3 of GridX. Use the selector to the left to change versions.

To find the object you're interested in, use the tree to the left...or use the quick links below. To print the documentation, simply navigate to the object of your choice and hit Print.

Main Classes of GridX

Grid
Pure GridX with only core modules.
GridCommon
A common GridX configuration which includes some useful modules.

Modules of GridX

HiddenColumns
Hide/show columns on demand.
GroupHeader
Grouped columns. Support arbitrarily complicated group structure.
PagedBody
Provide "load more" button in grid body. Suitable for mobile devices as well as desktop browser.
HeaderMenu
Add dropdown menus to column header.
IndirectSelectColumn
Provide a check-box/radio-button column for row selection.
Edit
Cell editing. Supports lazy save, undo/redo.
Tree
Tree grid. Supports child level lazy-loading, sorting and selection.
extendedSelect/Row
Select row using mouse or keyboard.
Support swipe-selection (optional), CTRL-selection, and SHIFT-selection.
Support unselectable row.

Supporting Widgets/Tools

QuickFilter
Filter the all filterable columns of grid in a text box.
LinkPager
Switch grid pages by link buttons.
Summary
Show total row count and selected row count.
exporter/toCSV
A function to export grid to CSV format.
